# Physics-Guided Spectral Decomposition (PGSD) for Chenda Drum Analysis

> **Companion repository for:**
> *"Physics-Guided Spectral Decomposition for Acoustic Analysis of Chenda and Shell Geometry Variants"*
> Submitted to ISMIR 2026 (anonymous review)

---

## What this repo contains

This repository provides all code, synthesised audio, figures, and notebooks needed to reproduce every result in the paper. No training data is required — all analysis is performed from first principles using the Fourier-Chebyshev membrane eigensolver.

---

## Quick start

```bash
git clone https://github.com/arcadehero3-eng/ismir2026
cd chenda-pgsd
pip install -r requirements.txt
python src/pgsd_v3.py          # runs all three tasks, prints results
python src/generate_eigenmodes.py  # regenerates eigenmode figures
```

## The Three diagnostic tasks

| Task | What PGSD recovers | Method | Result |
|------|--------------------|--------|--------|
| T1 | Membrane tension (N/m) | f₁ scaling law | Mean error 0.32–4.41% |
| T2 | Strike position (r/R) | Signal-level template matching | 0.3%R (sub-mm) |
| T3 | Skin condition index (0–100) | T60 from bandpass envelope | MAE = 0, monotone |

## Real recordings

Place your own Chenda WAV recordings in `audio/real_recordings/`.
Then run the analysis notebook:

```bash
jupyter notebook notebooks/real_recordings_analysis.ipynb
```

See `docs/AUDIO_GUIDE.md` for recording tips to maximise R² fit quality.

**Expected R² values:**
- R² > 0.85 → excellent, publish as-is
- R² 0.65–0.85 → good, mention model-reality gap
- R² < 0.50 → re-record (likely background noise or multiple strikes)

## Spectrogram generation

The notebook `notebooks/fft_synthesis_baseline.ipynb` generates spectrograms for all synthesised signals automatically. Figures are saved to `figures/spectrograms/`.

To generate spectrograms for your own recordings:

```python
from src.pgsd_v3 import *
import matplotlib.pyplot as plt
from scipy.signal import spectrogram

fs, data = wavfile.read("audio/real_recordings/your_strike.wav")
f, t, Sxx = spectrogram(data, fs=fs, nperseg=512, noverlap=448)
plt.pcolormesh(t, f[:200], 10*np.log10(Sxx[:200]+1e-10),
               shading='gouraud', cmap='magma')
plt.ylabel('Frequency (Hz)')
plt.xlabel('Time (s)')
plt.colorbar(label='Power (dB)')
plt.savefig("figures/spectrograms/my_strike.png", dpi=150)
```

```bash
# Table 2: Tension estimation

python src/pgsd_v3.py --task position

# Table 4: Skin condition
python src/pgsd_v3.py --task skin

# Table 5: Shape identification
python src/pgsd_v3.py --task shape

python src/baselines.py
```

Or run everything at once:
```bash
python src/pgsd_v3.py  # runs all tasks by default
```

## Physics model parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| R | 0.125 m | Membrane radius |
| T_ref | 3500 N/m | Reference tension |
| σ_ref | 1.4 kg/m² | Surface density |
| N | 18 modes | Number of eigenmodes |
| η₁ | 0.80 | Rayleigh viscous damping |
| η₂ | 8.8 × 10⁻⁶ | Rayleigh structural damping |
| N_r | 25 | Chebyshev nodes (radial) |
| N_θ | 24 | Fourier nodes (angular) |
